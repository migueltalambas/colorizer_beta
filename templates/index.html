Colorizer

A free, responsive way to turn black & white images and videos to color.

Get Started

About

Colorizer is powered by Generative Adversarial Networks, or GANs for short, an approach to generative modeling using deep learning methods, in this case convolutional neural networks. This is an open source project. It is free to study, and modify this code. Github Project Repo.
